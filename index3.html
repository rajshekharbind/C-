Toaster demo — single HTML (responsive + animated)
Uses your provided Sonner CSS and small JS to make toasts functional.

Usage

Click the buttons above to create toasts. Change position, theme or enable RTL to see how CSS variables switch layout and margins. To customize, edit the CSS variables on the [data-sonner-toaster] element in the JS.

Accessibility

Toasts are inserted with role="status" aria-live="polite" so screen readers are notified. Close buttons are keyboard-focusable.

Built-in animations, responsive mobile layout, and simple JS controls. Swipe-to-dismiss not implemented in this demo, but can be added.
